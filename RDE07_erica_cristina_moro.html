Branches o que é ?

O branch (ramos) no Git são semelhan tes a um ramo de uma árvore, onde o tronco seria a base do código. Desse modo é possível criar diversos ramos e fazer alterações, enquanto a base permanece intacta. O branch principal é denominado de main (master, na versão antiga) permitindo ao desenvolvedor alterar e corrijir a sua pagína com segurança.

Pode-se criar um branch através do branch-padrão do repositório, nele poderá trabalhar nesse novo branch e não irá alterar o branch-padrão.
